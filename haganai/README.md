# 僕は友達が少ない

## STORY

在学校总是独来独往的羽濑川小鹰，在一次偶然的情况下，目击到总是摆出一副臭脸的美少女三日月夜空一个人很开心地自言自语。

小鹰和夜空聊了一下该怎么交朋友的话题，没想到夜空竟然凭着莫名其妙的行动力成立一个以交朋友为目的的问题社团。

而且也不知道是哪里搞错，问题美少女们还接二连三地申请入社！

大家卯起来打美少女游戏电玩，一起去游泳、一起演戏。似乎一直没有抓对要领的他们，果真能交到朋友吗？

# 魔法少女リリカルなのは The MOVIE 2nd A's

![poster](poster.jpg)

## STORY

本作的时间点被设定于前作普蕾希亚·泰斯塔罗沙事件结束的数个月后。
在乘坐着轮椅、名为八神疾风的少女所有的一本古老精装书突然觉醒之时，从异样的光芒中出现的是……！

2005年放映的动画'魔法少女奈叶A's'的电影化、2010年放映的电影'魔法少女奈叶 The MOVIE 1st'的再续集。

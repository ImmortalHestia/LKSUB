# 魔弾の王と戦姫 

![poster](poster.jpg)

## STORY

吉斯塔特王国的统治体系由国王和七名战姬组成，七位战姬各自统领七块领地，挥舞着超乎想像的强大武器『龙具』驰骋在战场上。布琉努王国的贵族少年堤格尔在与吉斯塔特王国的战争中碰巧遇见了其中一位战姬，「银闪的风姬」艾莲。艾莲对堤格尔高超的弓箭技巧一见钟情，竟然对他说「从今以后你就是我的人了」并将他带往自己的领地……

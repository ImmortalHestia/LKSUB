## STORY

「Unhappy♪」是由琴慈创作的漫画，故事主要围绕聚集了背负着不幸的学生的天之御船学园1年7班展开，该班的学生云雀在开学第一天遇见了倒霉到底的 华子及极其不健康的牡丹，并且和她们一同试图寻找能让大家的高中生活变得“幸福”的方法。这样一部题材较为新颖、全员可怜的不幸喜剧，想必会引来不少人的关注。原作单行本至今已推出了4卷。

# いつか天魔の黒ウサギ

## STORY

成天无所事事的宫阪高中一年级学生·铁大兔，在幼时与拥有强大魔力的上古魔法师·赛托希梅亚相遇。两人订下契约后，大兔被希梅亚在体内注入“毒”，得到了“只要15分钟内不被连续杀7次就不会死”的诅咒。由此，这段“活七次遇见你”的颠覆校园奇幻故事正式揭幕……

# 月刊少女野崎くん

![poster](poster.jpg)

## STORY

「月刊少女野崎君」的故事从高中女生佐仓千代好不容易提起勇气向同年级的野崎梅太郎告白开始，可是此后她得到的却是野崎的亲笔签名，以及"要不要来我家？"的邀请。佐仓虽然对这意料之外的展开感到困惑，却还是带着期待来到野崎家，没想到等着她的却是漫画原稿，不知不觉间就顺着野崎的指示开始进行图黑作业。到这时佐仓才发现野崎是知名少女漫画家梦野咲子！

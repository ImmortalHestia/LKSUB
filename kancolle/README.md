# 艦隊これくしょん -艦これ-

![poster](poster.jpg)

## STORY

电视动画《舰队Collection》改编自娇喘游戏开发、DMM.com营运的同名黄赌毒网页游戏。

动画以驱逐舰吹雪为主人公，以她的视点展开故♀事。

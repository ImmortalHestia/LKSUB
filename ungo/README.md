# UN-GO

![poster](poster.jpg)

## STORY

被人称为最后的名侦探结城新十郎，与他的拍档因果，组成了要解明这世上之谜及真实的侦探二人组合。在面对常理不能解释的谜题或案件时，凭着新十郎的推理及因果的协力，这些难题往往也会迎刃而解。但是，当新十郎对案件越发思考之时，他便发现某些地方的不对劲，并随着他的深入了解，他和因果更渐渐与某件大事件扯上关系。

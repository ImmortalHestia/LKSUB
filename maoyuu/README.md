# まおゆう魔王勇者

![poster](poster.jpg)

## STORY

魔王和勇者之间。正统王道RPG的对决——

一边是，自由操控剑和魔法、一夫当关万夫莫开的勇者。一边是，统治魔界一切的魔王。

在魔界耸立的魔王城中，魔王接见了勇者。

魔王给出了不可思议的提议：“成为我的东西吧。”

勇者明白无误的回答：“不!”

为了人界和魔界，这样的两人开始挑战改革这个世界……

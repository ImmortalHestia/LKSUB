## STORY

国中时期曾获得最优秀选手奖的男主角长谷川昴，因为篮球社社长的关系而使他在高中的篮球活动被禁止，正当他灰心之际，他那在国小当老师的阿姨篁美星介绍他到私立慧心学园初等部教萝莉打蓝球。

五位性格各异的萝莉喊着哥哥、哥哥，教练、教练的篮球生活开始了。。。

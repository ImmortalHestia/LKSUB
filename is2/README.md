# IS 〈インフィニット・ストラトス〉 2

![poster](poster.jpg)

## STORY

只對女性有反應、世界最強的超級兵器“nfinite Stratos（IS）”。

因為IS的問世，性別平等被破壞，世界變成女尊男卑的社會。主角“織斑一夏”在高中的入學試驗誤入IS操縱者育成學校“IS學園”的試驗會場並意外地啟動了IS。成為全世界關注的“世界上唯一能操縱IS的男性”的一夏，被政府強制就讀向來只有女性的IS學園，在全校師生對他充滿好奇心的視線下，一夏波瀾壯闊校園生活就此展開。
